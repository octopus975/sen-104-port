To Do List

Project 2

Login And Register

About This Project This Login and Register page is part of a user authentication system designed to securely manage user access. The registration form allows new users to sign up by providing their details, while the login form lets existing users access their accounts. The project is built using HTML, CSS, and JavaScript, and may include backend integration with a database or authentication service (e.g., Firebase, PHP, or Node.js). It emphasizes responsive design, form validation, and user experience. This project helped me understand how to structure secure forms and manage user sessions.

Project 3

Calculator

About This Project This Calculator website is a simple web-based tool that performs basic arithmetic operations like addition, subtraction, multiplication, and division. It’s built using HTML, CSS, and JavaScript, with a focus on a clean layout and responsive design. The goal of this project is to understand how user input can be captured and processed dynamically in a browser. Through this project, I gained hands-on experience in handling events, updating the display in real-time, and building a fully functional utility with JavaScript.

Project 4

Clone Netflix

About This Project This Netflix clone is a web application that mimics the design and basic functionality of Netflix. It displays a list of movies or TV shows fetched from an API (like TMDB) and allows users to browse content by category. The app is built using modern web technologies such as HTML, CSS, JavaScript, and optionally React.js for a more dynamic experience. Some features include a responsive UI, video thumbnails, hover effects, and a navigation bar. This project helped me practice working with APIs, component-based design, and real-time data rendering.

Project 5

Responsive Coffee

About This Project "Responses Coffees" is a responsive coffee shop website designed to provide users with an engaging experience across all devices. The site features a modern layout showcasing different coffee products, descriptions, prices, and user interactions like ordering or viewing details. Built using HTML, CSS, and JavaScript (and optionally Bootstrap or React), the site adjusts seamlessly for mobile, tablet, and desktop screens. This project helped me understand responsive design principles, layout structuring, and interactive UI development.
